## 🚀 如何使用

1. 克隆项目代码到本地：

   ```bash
   git clone https://github.com/yourusername/yourproject.git
   ```

2. 进入项目目录，安装依赖：

   ```bash
   cd yourproject
   pnpm install
   ```

3. 运行项目：

   ```bash
   pnpm dev
   ```

   此时，项目将运行在 `http://localhost:5173` 上。
